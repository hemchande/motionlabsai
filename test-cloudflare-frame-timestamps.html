Cloudflare Stream Frame Timestamps Test

This script tests loading a Cloudflare Stream video and extracting frame timestamps corresponding to image frames.

Instructions:

  1. Enter a Cloudflare Stream video URL or ID
  2. Click "Load Video" to initialize the player
  3. Use controls to play/pause and seek through the video
  4. Click "Capture Frame Timestamp" to get current frame info
  5. Click "Start Frame Analysis" to analyze frame timestamps
